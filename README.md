# COPE Browser Extension

## Overview

COPE is a browser extension designed to support interactions with large language model platforms such as ChatGPT and Perplexity. The project is provided for research and academic purposes.

## Installation

Clone the repository and install dependencies:

```bash
npm install
```

Build the extension using:

```bash
npm run build
```

After the build process is completed, a `dist` directory will be generated.

To use the extension:

1. Open the Chrome Extensions page:

```text
chrome://extensions/
```

2. Enable **Developer Mode**
3. Click **Load unpacked**
4. Select the generated `dist` folder

The extension will then be installed locally in the browser.

## Compatibility

This extension was developed and tested using the ChatGPT and Perplexity web interfaces available as of December 17, 2025.

Because both platforms are under active development, future interface or API updates may introduce breaking changes that affect the functionality of the extension. Additional maintenance or adaptation may therefore be required for continued compatibility.
